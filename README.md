# 🎲 Dice Roller Program

A simple and interactive Dice Roller application built with **HTML, CSS, and JavaScript**. Users can choose how many dice they want to roll, and the application generates random dice values while displaying the corresponding dice images.

## ✨ Features

- 🎲 Roll one or multiple dice at the same time
- 🔢 Generates random numbers from **1 to 6**
- 🖼️ Displays dice images based on the rolled values
- ⚡ Instant results without refreshing the page
- 📱 Responsive and easy-to-use interface
- 🎨 Clean and modern UI

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 How It Works

1. Enter the number of dice you want to roll.
2. Click the **Roll Dice** button.
3. JavaScript generates random numbers between **1 and 6** using `Math.random()`.
4. The rolled numbers are displayed on the screen.
5. Matching dice images are dynamically generated and displayed.
